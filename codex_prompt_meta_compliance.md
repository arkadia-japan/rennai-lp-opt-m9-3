# Meta広告ポリシー準拠のためのLP修正プロンプト

## 修正対象ファイル
`index.html`

## 修正が必要な理由
Meta広告のポリシーに準拠するため、以下の表現を修正する必要があります：

### 1. 断定表現・最上級表現の修正
- **問題箇所**: 「100％」「最短最速」「10倍」「月に10人以上」「半年で10人以上」
- **理由**: Meta広告では断定表現や最上級表現、非現実的な結果の訴求が厳しく制限されている

### 2. 性的な関係を暗示する表現の修正
- **問題箇所**: 「250人以上の女性と関係を持ちました」「一夜限りの関係」「女性と関係構築」
- **理由**: 性的なものを暗示・示唆するコンテンツはMeta社のポリシーで厳しく制限されている

## 具体的な修正指示

### 修正1: 第7話のタイトル（118行目）
```html
<!-- 修正前 -->
<h3>第7話：本命女性と付き合う確率を100％にするたった1つの方法</h3>

<!-- 修正後 -->
<h3>第7話：本命女性と付き合う可能性を大幅に高めるたった1つの方法</h3>
```

### 修正2: 第2話の内容（62行目）
```html
<!-- 修正前 -->
<li>マッチングアプリの反応が10倍に跳ね上がった、コンサル生の外見改善事例</li>

<!-- 修正後 -->
<li>マッチングアプリの反応が大幅に向上した、コンサル生の外見改善事例</li>
```

### 修正3: 第3話のタイトル（66行目）
```html
<!-- 修正前 -->
<h3>第3話：恋愛経験が少ない男でも月に10人以上の女性とデートできる出会い戦略</h3>

<!-- 修正後 -->
<h3>第3話：恋愛経験が少ない男でも多くの女性とデートできる出会い戦略</h3>
```

### 修正4: 第5話の内容（100行目）
```html

<!-- 修正後 -->
<li>最小限の努力で、効率的に本命彼女を作るための戦略とは</li>
```

### 修正5: プロフィール文（153行目）
```html
<!-- 修正前 -->

<!-- 修正後 -->
<p>アプリやストリートで、多くの女性との出会いを経験した後、1人の愛する女性と出会い結婚。</p>
```

### 修正6: 実績例（158行目）
```html
<!-- 修正前 -->
<li>童貞から半年で10人以上の女性と関係構築、理想の彼女をGET（25歳・会社員）</li>

<!-- 修正後 -->

```

## 修正のポイント
1. **数値の具体化を避ける**: 「100％」「10倍」「10人以上」→「大幅に」「多くの」「複数の」
2. **断定表現を和らげる**: 「必ず」「絶対に」「確実に」→「可能性を高める」「効果的に」
3. **性的暗示を排除**: 「関係を持つ」「関係構築」→「出会いを経験」「良好な関係を築く」
4. **最上級表現を修正**: 「最短最速」→「効率的に」

## 注意事項
- 修正後も元の意図やメッセージ性は保持する
- 読みやすさと自然さを維持する
- 他の部分との整合性を確保する

この修正により、Meta広告のポリシーに準拠しながら、LPの訴求力を維持できます。
