次回からのPDF変換手順

  ステップ1: PDFファイルを配置

  変換したいPDFを以下のフォルダに入れます：
  C:\Users\yoona\99.Project\kindle PDF-Inbox

  ステップ2: PowerShellを開く

  変換スクリプトがあるフォルダに移動：
  cd C:\Users\yoona\99.Project\docling-main

  ステップ3: 変換実行

  方法A: 全PDFを一括変換

  python convert_all_pdfs.py
  - kindle PDF-Inbox内の全PDFファイルを自動変換
  - 既に変換済みのファイルは自動スキップ

  方法B: 特定のPDFのみ変換

  1. convert_specific_pdfs.pyをメモ帳で開く
  2. target_filesリストを編集：
  target_files = [
      '新しいファイル1.pdf',
      '新しいファイル2.pdf'
  ]
  3. 保存して実行：
  python convert_specific_pdfs.py

  変換後のファイル

  以下のフォルダに保存されます：
  C:\Users\yoona\99.Project\kindle markdown

  1. PDFをkindle PDF-Inboxに入れる
  2. cd C:\Users\yoona\99.Project\docling-main
